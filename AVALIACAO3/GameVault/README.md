# Game Vault

Aplicación para catalogar juegos consumiendo una API.

## Requisitos

- Node.js
- Expo CLI

## Instalación

1. Clonar el repositorio.
2. Instalar dependencias:
   ```bash
   npm install
   ```

## Ejecución

```bash
npx expo start
```

## Funcionalidades

- Listado de juegos
- Búsqueda de juegos
- Creación de juegos
- Edición de juegos
- Eliminación de juegos

## Estructura

- `app/`: Pantallas y navegación (Expo Router).
- `servicios/`: Lógica de comunicación con la API.
- `componentes/`: Componentes reutilizables (si los hubiera).

## Notas

- Las variables y funciones están en español.
- Se utiliza la API: http://177.44.248.50:8080
