# Aula 1 - Introdução ao React Native

## Objetivo
Criar seu primeiro aplicativo React Native e entender os conceitos básicos.

## Exercícios
1. Criar um novo projeto React Native
2. Modificar o componente principal
3. Adicionar estilização básica
4. Criar componentes personalizados
