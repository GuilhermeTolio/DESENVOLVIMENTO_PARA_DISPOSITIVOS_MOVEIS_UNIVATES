# Aula 2 - Navegação e Roteamento

## Objetivo
Implementar navegação entre telas usando Expo Router.

## Exercícios
1. Configurar navegação por tabs
2. Criar múltiplas telas
3. Implementar navegação por stack
4. Passar parâmetros entre telas

## Recursos
- [Expo Router](https://docs.expo.dev/router/introduction/)
- [React Navigation](https://reactnavigation.org/)

## Entrega
- [ ] Aplicativo com navegação funcional
- [ ] Pelo menos 3 telas diferentes
- [ ] Navegação por tabs implementada
