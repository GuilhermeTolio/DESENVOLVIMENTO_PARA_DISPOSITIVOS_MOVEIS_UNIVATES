# Test 1 - Aplicativo de Lista de Tarefas

## Descrição
Desenvolva um aplicativo de lista de tarefas (To-Do App) completo usando React Native.

## Requisitos
1. **Adicionar Tarefa**: Interface para adicionar novas tarefas
2. **Lista de Tarefas**: Visualizar todas as tarefas
3. **Marcar como Concluída**: Funcionalidade para marcar tarefas como feitas
4. **Excluir Tarefa**: Remover tarefas da lista
5. **Persistência**: Salvar dados localmente

## Critérios de Avaliação
- Funcionalidade (40%)
- Interface de usuário (30%)
- Código limpo e organizado (20%)
- Criatividade e extras (10%)
